Server setup
ACTION ML adds 8 new server variables, that can be set, to enhance the game.

bb_fix   1 | 0
Turn on/off boundingbox sizes from AQ:G (default 1)

locationlimit   n
Max read locations from locationfiles (default 40)

m4script_check   0 | 1 | 2
M4script detection, off/log only/log & autokick (default 2, autokicking)

video_check   1 | 0
Turn on/off gl_modulate checking (default 1)

video_force_restart   1 | 0
Turn on/off a forced vid_restart on the client each map (default 0)
(use this for a more strict check)

video_max_3dfx   n
Max allowed gl_modulate value for 3dfx cards (default 1.5)

video_max_3dfxam   n
Max allowed gl_modulate value for (amd) 3dfx cards (default 1.5)

video_max_opengl   n
Max allowed gl_modulate value for openGL cards (default 3.0)
