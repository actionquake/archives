Feature list

The following is a brief list of all the features in FireTeams

  • New Server Variable - accurate_sniper - New for V1.35
  • Use of %L commands in say_team messages - Adds location info to your messages - New for V1.3
  • New client command - team (displays team information)
  • Skin Indicator
  • New Server Variable - max_teams
  • New Server Variable - players_per_team
  • New Server Variable - match_countdown
  • Dynamic Team Creation
  • New Menu system for joining and creating teams
  • New Scoreboards for both Deathmatch and Teamplay
  • Two modes of play - Deathmatch Teams, and Teamplay
  • New Server Variable - deathmatchmode
