3. New server variables

# more splatter
The variable ' more splatter ' can have the values 0, 1 or 2.
A value of 1 lets parts of the body appear with Sniper hits
and a value of 2 makes also parts of the body active with shells.

B2K (fount to of Kill) is a AQ clan, which enjoys of ever larger popularity.
By that set this variable to ' 1? becomes instead of the normal MOTD
(Message OF The Day) the B2K-MOTD displayed. In the play nothing changes.

# blowoncontact
Shells in AQ always explode after a certain time,
which is naturally realistic. In addition, it makes fun with shells around itself to throw,
those with body contact explode. The variable must on?1? are set,

4. Further server and Client instruction

* msgbox
This instruction needs two parameters. On the one hand a text, which is displayed
and on the other hand a password is. Server administrators know the password under
the eMail address inquire ReDCoMeT@rivalnet.de. If one e.g. into the server console
msgbox " server shutdown in 30 seconds " < passwords > inputs, becomes this message
all players in the center of the display in green displayed. Becomes no password
input, nothing occurs. In later versions further Admin instruction become
available its.

* admin_msgbox
Functioned exactly the same as the server instruction msgbox. The only difference is,
that this instruction can be used also by Clients. Becomes natural also in
this case needs a password.

* zoom shot
This instruction is very useful for all Sniperliebhaber. One can do thereby those
Sniper Rifle with an instruction on a certain zoom shot and sensitivity level set.
E.G.: zoom shot 4 5 would set the Sniper Rifle on zoom shot level 4 and on sensitivity level 5.
